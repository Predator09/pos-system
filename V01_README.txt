SmartStock — V01 portable folder
================================

WHAT TO COPY
  Copy this entire V01 folder to the other PC (USB, network, zip).
  Do not move only SmartStock.exe — keep the _internal folder next to it.

HOW TO RUN
  Double-click SmartStock.exe. Python is not required on the new PC.
  On the first run you will be asked for the installation code once.

WHERE DATA IS STORED
  Shop database and settings:  %LOCALAPPDATA%\SmartStock

IF THE APP WON'T START
  Install "Microsoft Visual C++ Redistributable 2015-2022 (x64)" from Microsoft.
